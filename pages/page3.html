Project Files
  • index.html: This page is a simple HTML page a with a single SVG tag. This tag is being manipulated using a javascript file(drawsvg.js)
  • drawsvg.js: java script that loads the home page and navigates to other pages as you click onthe menu.
  • style.css: simple styling for the page.
  • json/conf.js: page branding information (logo, slogan).
  • json/menu.js: defines menu items and links.
  • pages/ folder that holds HTML files for sub pages contents.
How to use it?
  • Download the latest package from here.
  • Extract in ina folder
  • Open index.html
  • Modify json files to reflect your design
  • Modify html pages to reflect your contents
For more information please visit the Wiki Documentation.
